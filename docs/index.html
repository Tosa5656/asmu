О проекте

Пять консольных утилит, которые закрывают типичные задачи по безопасности: SSH, логи, пароли, сеть и справочник по атакам. У каждой команды есть help — просто запустите smssh help, smlog help и т.д.

Утилита
Назначение
smssh
Безопасность SSH: конфиг, ключи, мониторинг атак
smlog
Логи: чтение, поиск, отчёты, мониторинг в реальном времени
smpass
Пароли: хранение с AES256, хэши SHA256
smnet
Сеть: порты, соединения, трафик, статистика
smdb
База MITRE ATT&CK: описание атак и рекомендации по защите

Утилиты

🔐

smssh

Проверка и усиление настроек SSH-сервера, разбор логов и поиск признаков атак. Анализ конфигурации, генерация безопасного конфига, мониторинг brute force и dictionary-атак.

Анализ конфига Мониторинг атак Генерация ключей
smssh analyze · smssh monitor · smssh gen-key
📋

smlog

Чтение, поиск и мониторинг логов, в том числе journald. Файловые логи и systemd journal, поиск по ключевым словам, топ IP и пользователей, готовые отчёты.

systemd journal Поиск Отчёты
smlog read · smlog search · smlog monitor
🔑

smpass

Локальное хранение паролей с шифрованием и хэшированием. Хранение с AES256, SHA256 для хэшей. Данные в ~/.asmu/storage/.

AES256 SHA256 Безопасное хранение
smpass add-password · smpass hash-sha256
🌐

smnet

Сканирование портов, активные соединения и статистика по интерфейсам. Список соединений TCP/UDP и процессов, мониторинг трафика на интерфейсе (pcap).

Сканирование портов Мониторинг Статистика
smnet scan · smnet connections · smnet stats
📚

smdb

Справочник техник MITRE ATT&CK с описаниями и подсказками, какими утилитами ASMU можно прикрыться. Список и поиск по техникам, описание и рекомендации по защите, указание инструментов ASMU.

MITRE ATT&CK Поиск Рекомендации
smdb list · smdb search <keyword> · smdb tools <attack_id>

Установка

Требования: Linux, компилятор с C++20, OpenSSL, libpcap. Опционально: MaxMind GeoLite2 для GeoIP в smssh.

  • Linux
  • Компилятор C++20
  • OpenSSL
  • libpcap
  • MaxMind GeoLite2
git clone https://github.com/Tosa5656/asmu
cd asmu
make
make check
make doc
sudo make install

Для мониторинга SSH через systemd:

sudo make install-systemd
systemctl enable smssh
systemctl start smssh
