Возможности

Всё для безопасности Linux

Комплексное решение для мониторинга и защиты серверов

Модульная архитектура

Пять независимых утилит, каждая решает свою задачу. Используйте только то, что нужно.

Высокая производительность

Написано на C++20 с фокусом на скорость и эффективность использования ресурсов.

Мониторинг в реальном времени

Отслеживайте атаки SSH, анализируйте логи и сетевой трафик в режиме реального времени.

Криптография

AES256 для шифрования, SHA256 для хэширования. Используем проверенные алгоритмы OpenSSL.

CLI и GUI

Консольные утилиты для автоматизации и графический интерфейс на Qt для удобной работы.

MITRE ATT&CK

Встроенная база знаний с описанием атак и рекомендациями по защите от них.

Утилиты

Пять инструментов безопасности

Каждая утилита решает конкретную задачу администратора

🔐

smssh

SSH Security

Анализ конфигурации SSH, генерация безопасных настроек, мониторинг атак brute force и dictionary, парсинг логов.

  • Анализ конфигурации
  • Генерация ключей
  • Мониторинг атак
  • Парсинг логов
smssh analyze
📋

smlog

Log Analysis

Чтение и анализ системных логов, поддержка journald, поиск по ключевым словам, готовые отчёты, мониторинг в реальном времени.

  • Файловые логи и journald
  • Поиск и фильтрация
  • Готовые отчёты
  • Real-time мониторинг
smlog monitor
🔑

smpass

Password Manager

Локальное хранение паролей с шифрованием AES256, генерация хэшей SHA256, безопасное хранение в ~/.asmu/storage/.

  • Шифрование AES256
  • Хэширование SHA256
  • Локальное хранение
  • Безопасный доступ
smpass add-password
🌐

smnet

Network Monitor

Сканирование портов, мониторинг активных соединений TCP/UDP, анализ трафика через pcap, статистика по интерфейсам.

  • Сканирование портов
  • Мониторинг соединений
  • Анализ трафика
  • Статистика интерфейсов
smnet scan
📚

smdb

MITRE ATT&CK

База знаний MITRE ATT&CK с описанием техник атак, рекомендациями по защите и привязкой к утилитам ASMU.

  • База MITRE ATT&CK
  • Поиск по техникам
  • Рекомендации по защите
  • Привязка к утилитам
smdb search brute

Установка

Быстрый старт

Установите ASMU за несколько минут

01

Требования

  • Linux
  • C++20 компилятор
  • OpenSSL
  • libpcap
02

Установка

bash 
git clone https://github.com/Tosa5656/asmu
cd asmu
make
make check
sudo make install
03

Готово!

Утилиты установлены в /usr/local/bin/

smssh help
smlog help
smpass help
smnet help
smdb help

GUI интерфейс 

make gui
sudo make install-gui
asmu-gui

Systemd мониторинг 

sudo make install-systemd
systemctl enable smssh
systemctl start smssh
