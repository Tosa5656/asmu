T1078 - Valid Accounts

Описание атаки

Использование валидных учетных данных для получения доступа к системам, приложениям или сетевым ресурсам. Это может быть результатом кражи учетных данных, использования стандартных/умолчательных паролей или компрометации легитимных аккаунтов.

Принцип работы

  • Кража учетных данных через фишинг
  • Использование найденных/угаданных паролей
  • Эксплуатация стандартных учетных записей
  • Pass-the-hash атаки
  • Использование украденных токенов/ключей

Защита с помощью ASMU

smpass - Password Management

  • Генерация безопасных паролей: smpass add-password
  • Хеширование: smpass hash-sha256, smpass hash-aes256
  • Хранение: Безопасное хранение учетных данных

smssh - SSH Security

  • Отключение password auth: smssh apply /etc/ssh/sshd_config
  • Мониторинг необычной активности: smssh monitor
  • Генерация ключей: smssh gen-key

smlog - Log Analysis

  • Анализ входов: smlog search "Accepted" /var/log/auth.log
  • Мониторинг подозрительной активности: smlog monitor

Рекомендации

  • Регулярная смена паролей
  • Многофакторная аутентификация (MFA)
  • Принцип наименьших привилегий
  • Мониторинг использования аккаунтов
  • Использование ключевой аутентификации
