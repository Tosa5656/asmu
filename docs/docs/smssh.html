smssh

Утилита и API для анализа конфигурации SSH, обнаружения атак и генерации ключей.

Назначение

Проверка и усиление настроек SSH-сервера, разбор логов и поиск признаков атак (brute force, dictionary, аномалии по GeoIP).

Исходники

  • smssh/smssh.cpp — точка входа, команды (help, analyze, generate, check, apply, show, monitor, parse-log, gen-key, post-config).
  • smssh/sshConfig.cpp, sshConfig.h — загрузка и разбор sshd_config, анализ безопасности, генерация безопасного конфига.
  • smssh/sshAttackDetector.cpp, sshAttackDetector.h — парсинг логов SSH и детекция атак.
  • smssh/smssh_config.h — константы конфигурации.

API (smssh_api.h)

Класс SecurityManager::SSHSecurity. Типы: SSHConfigIssue, SSHConnectionAttempt, SSHAttackAlert, SSHSecurityRecommendation, SSHSecurityReport. Результаты обёрнуты в SSHResult<T> с кодами SSHError.

Основные методы: analyzeConfiguration(), applySecurityHardening(), generateSecureConfig(), detectAttacks(), monitorAttacks(), stopMonitoring(), generateKeyPair(), getServerStatus(), testConnectivity().
