smpass

Утилита и API для шифрованного хранения паролей и хэширования (AES256, SHA256).

Назначение

Локальное хранение паролей с шифрованием и хэшированием. Хранение с AES256, SHA256 для хэшей. Данные в ~/.asmu/storage/.

Исходники

  • smpass/smpass.cpp — точка входа (add-password, delete-password, hash-sha256, hash-aes256).
  • smpass/storage.cpp, storage.h — работа с хранилищем записей.
  • smpass/aes256.h, sha256.h — крипто-примитивы (часто поверх OpenSSL).

Команды CLI

КомандаОписание
smpass helpСправка
smpass add-passwordИнтерактивно добавить запись (сервис, логин, пароль)
smpass delete-passwordУдалить запись
smpass hash-sha256 "строка"Вывести SHA256-хэш строки
smpass hash-aes256 "строка"Шифрование строки AES256 (с выводом в формате хранилища)

API (smpass_api.h)

Класс SecurityManager::PasswordManager. Типы: HashAlgorithm (SHA256, AES256), PasswordEntry. Результаты — PasswordResult<T> с кодами PasswordError.

Методы: hashString(), addPassword(), getPassword(), deletePassword(), listEntries() и др. Подробности в api/include/smpass_api.h.
