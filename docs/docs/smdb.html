smdb

Утилита и API для работы с базой знаний MITRE ATT&CK: описание атак и рекомендации по защите с помощью ASMU.

Назначение

Справочник техник с описаниями и подсказками, какими утилитами ASMU можно прикрыться. Список и поиск по техникам, описание и рекомендации по защите, указание инструментов (smssh, smlog и др.). Документация по атакам — HTML в doc/attacks/, при установке — в /usr/share/doc/asmu/attacks/.

Исходники

  • smdb/smdb.cpp — точка входа и логика: чтение HTML из doc/attacks/, извлечение заголовков и контента (getHtmlTitle, getHtmlContent), поиск по ключевым словам, вывод списка атак и рекомендаций (list, search, show, tools).

Команды CLI

КомандаОписание
smdb helpСправка
smdb listСписок всех атак (ID и название)
smdb search <keyword>Поиск по ключевым словам
smdb show <attack_id>Подробная информация об атаке (например T1110)
smdb tools <attack_id>Команды утилит ASMU для защиты от атаки

API (smdb_api.h)

Класс Asmu::AttackDatabase. Типы: AttackInfo, ProtectionGuidance, AttackSearchResult, DatabaseStats. Результаты — DatabaseResult<T> с кодами DatabaseError.

Методы: searchAttacks(), getAttackInfo(), getProtectionGuidance(), getAttacksByTactic(), getAttacksByPlatform(), listAllAttacks(), getDatabaseStats(), exportAttackInfo(), getRelatedAttacks(), getMitigationStrategies().
