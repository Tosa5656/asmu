smnet

Утилита и API для сканирования портов, мониторинга соединений и статистики по интерфейсам.

Назначение

Список соединений (TCP/UDP) и процессов, мониторинг трафика на интерфейсе (pcap), статистика по интерфейсам и протоколам.

Исходники

  • smnet/smnet.cpp — точка входа (scan, connections, stats).
  • smnet/portScanner.h — сканирование портов.
  • smnet/trafficMonitor.h — мониторинг трафика (pcap).
  • smnet/networkStats.h — статистика по интерфейсам.

Команды CLI

КомандаОписание
smnet helpСправка
smnet scanОткрытые порты и соединения
smnet connections [интерфейс]Активные соединения (по умолчанию eth0)
smnet statsСтатистика интерфейсов

API (smnet_api.h)

Класс SecurityManager::NetworkMonitor. Типы: PortResult, ConnectionInfo, InterfaceInfo. Результаты — NetworkResult<T> с кодами NetworkError.

Методы: scanPorts(), getActiveConnections(), getInterfaceStats() и др. Подробности в api/include/smnet_api.h.
