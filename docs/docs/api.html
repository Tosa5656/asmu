C++ API

Программный доступ ко всем возможностям ASMU из кода на C++. Единая точка входа — securitymanager.h.

Подключение и инициализация

#include <securitymanager.h>

int main() {
    SecurityManager::initialize();

    SecurityManager::PasswordManager pwd_mgr;
    auto hash = pwd_mgr.hashString("password", SecurityManager::HashAlgorithm::SHA256);

    SecurityManager::NetworkMonitor net_mgr;
    auto ports = net_mgr.scanPorts();

    SecurityManager::AttackDatabase attack_db;
    auto attacks = attack_db.searchAttacks("brute force");

    SecurityManager::cleanup();
    return 0;
}

Результаты и ошибки

Каждый модуль возвращает обёртки вида XxxResult<T> с полями code, message, data и методом success(). Коды ошибок — перечисления XxxError (SUCCESS, FILE_NOT_FOUND, PERMISSION_DENIED и т.д.).

Сборка

make libasmu.a   # статическая библиотека
make all         # утилиты + библиотека

Зависимости: OpenSSL, libpcap, libmaxminddb (опционально), C++20.
