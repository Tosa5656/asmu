Документация ASMU

Advanced Security Manager Utils — полное руководство по использованию, архитектуре и API проекта.

О проекте

ASMU — это набор профессиональных консольных утилит для администраторов Linux, предназначенных для мониторинга, анализа и усиления безопасности серверов. Проект написан на современном C++20 и предоставляет как CLI-интерфейс, так и графическую оболочку на Qt.

Ключевые особенности

  • Модульная архитектура — каждая утилита решает конкретную задачу
  • Высокая производительность — написано на C++20 с фокусом на скорость
  • Единый API — статическая библиотека для интеграции в свои проекты
  • Криптография — AES256, SHA256 через OpenSSL
  • MITRE ATT&CK — встроенная база знаний по атакам и защите

Утилиты

🔐 smssh Безопасность SSH: анализ конфигурации, генерация ключей, мониторинг атак brute force и dictionary 📋 smlog Анализ логов: файловые логи и journald, поиск, отчёты, мониторинг в реальном времени 🔑 smpass Менеджер паролей: локальное хранение с шифрованием AES256, генерация хэшей SHA256 🌐 smnet Сетевой мониторинг: сканирование портов, анализ соединений TCP/UDP, статистика трафика 📚 smdb База знаний: MITRE ATT&CK с описанием техник атак и рекомендациями по защите

Быстрый старт

Требования

  • Linux (любой современный дистрибутив)
  • Компилятор с поддержкой C++20 (GCC 10+, Clang 11+)
  • OpenSSL (для криптографии)
  • libpcap (для сетевого мониторинга)
  • Qt5 или Qt6 (опционально, для GUI)
  • MaxMind GeoLite2 (опционально, для GeoIP в smssh)

Установка

git clone https://github.com/Tosa5656/asmu
cd asmu
make
make check    # запустить тесты
sudo make install

Утилиты будут установлены в /usr/local/bin/, документация — в /usr/share/doc/asmu/.

Сборка GUI

make gui
sudo make install-gui
asmu-gui

Сборка API библиотеки

make libasmu.a

Архитектура

Проект организован в модульную структуру, где каждая утилита является независимым модулем с собственной логикой, но все они используют общие компоненты:

  • logger — единая система логирования
  • argsparser — парсинг аргументов командной строки
  • api — C++ API для всех модулей
  • json — работа с JSON (nlohmann/json)

Подробнее об архитектуре читайте в разделе Архитектура.

C++ API

ASMU предоставляет статическую библиотеку libasmu.a с единым API для всех модулей. Это позволяет интегрировать функциональность ASMU в свои C++ проекты.

Основные классы API:

  • PasswordManager — управление паролями с шифрованием
  • NetworkMonitor — сетевой мониторинг и анализ
  • LogAnalyzer — анализ системных логов
  • SSHSecurity — проверка безопасности SSH
  • AttackDatabase — работа с базой MITRE ATT&CK

Подробная документация API доступна в разделе C++ API.
