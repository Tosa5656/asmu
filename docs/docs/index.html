Документация ASMU

Advanced Security Manager Utils — описание модулей, архитектуры и API проекта.

Введение

ASMU — набор консольных утилит для администраторов Linux: мониторинг, анализ и усиление безопасности. Проект написан на C++20 и состоит из пяти основных утилит и единого C++ API.

Модули

smssh
Безопасность SSH: конфиг, ключи, мониторинг атак smlog
Логи: чтение, поиск, отчёты, journald smpass
Пароли: AES256, SHA256, хранилище smnet
Сеть: порты, соединения, трафик smdb
База MITRE ATT&CK и рекомендации

Сборка и зависимости

Требования: Linux, компилятор C++20, OpenSSL, libpcap. Опционально: MaxMind GeoLite2 для GeoIP в smssh.

make          # утилиты
make libasmu.a  # статическая библиотека API
make check   # тесты
make doc     # Doxygen-документация

Связанные разделы

  • Архитектура — дерево каталогов и связи модулей.
  • C++ API — использование из кода (PasswordManager, NetworkMonitor, LogAnalyzer, SSHSecurity, AttackDatabase).
  • Общие компоненты — logger и argsparser.
